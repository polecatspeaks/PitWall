## ATLAS Core Principles

### What Makes ATLAS Exceptional

**1. Data Density Without Clutter**
- Multiple channels visible simultaneously
- Clear parameter labeling with units
- Synchronized cursors across all traces
- Color-coded traces (red vs blue for comparison)

**2. Professional Engineering Focus**
- Precise numerical values (not just graphs)
- Scientific notation and proper units
- Tabular data alongside visualizations
- Export and analysis tools

**3. Hierarchical Organization**
- Workbook â†’ Pages â†’ Displays
- Logical grouping of related parameters
- Quick navigation between views
- Customizable layouts

**4. Real-Time + Historical**
- Live telemetry streaming
- Historical data playback
- Lap-by-lap comparison
- Session management

### What We'll Improve for Sim Racing

**1. Modern Visual Design**
- Contemporary F1 broadcast aesthetics
- High-contrast dark theme
- Animated transitions (subtle)
- Touch-friendly controls

**2. Accessibility**
- Larger fonts for at-a-glance reading
- Color-blind safe palettes
- Keyboard shortcuts
- Voice command integration

**3. Gaming Integration**
- In-game overlay mode
- Quick pit strategy buttons
- AI engineer chat interface
- Real-time alerts and warnings

**4. Simplified Workflow**
- One-click session start
- Auto-save and replay
- Preset workbook templates
- Guided setup wizard

### Key ATLAS Features Implemented

**1. Synchronized Cursor**
- Vertical line across all traces
- Real-time value table updates
- Click/drag or arrow keys to scrub

**2. Lap Comparison**
- Red trace = Current lap (LAP 15)
- Blue trace = Reference lap (LAP 12 personal best)
- Delta column shows differences

**3. Scientific Parameter Naming**
- Prefix indicates type:
  - `v` = Velocity
  - `n` = Normalized (0-100%)
  - `t` = Temperature
  - `r` = Radians
  - `f` = Fuel
  - `g` = G-force
- Consistent with ATLAS convention

**4. Multi-Channel Display**
- Speed, throttle, brake, steering, temps all stacked
- Aligned time axis
- Individual Y-axis scales
- Collapsible channels [â–¼] button

**5. Data Table**
- Exact values at cursor position
- Delta calculations with trend arrows
- Unit labels
- Sortable columns

// Using LiveChartsCore for ATLAS-style telemetry traces
public class AtlasWaveformChart : UserControl
{
    private CartesianChart _chart;
    
    public AtlasWaveformChart()
    {
        _chart = new CartesianChart
        {
            Series = new ISeries[]
            {
                new LineSeries<TelemetryPoint>
                {
                    Name = "Current Lap",
                    Values = CurrentLapData,
                    Stroke = new SolidColorPaint(SKColors.Red) { StrokeThickness = 2 },
                    Fill = null,
                    GeometrySize = 0
                },
                new LineSeries<TelemetryPoint>
                {
                    Name = "Reference Lap",
                    Values = ReferenceLapData,
                    Stroke = new SolidColorPaint(SKColors.Cyan) { StrokeThickness = 2 },
                    Fill = null,
                    GeometrySize = 0
                }
            },
            
            XAxes = new[]
            {
                new Axis
                {
                    Name = "Distance (m)",
                    TextSize = 12,
                    LabelsPaint = new SolidColorPaint(SKColors.LightGray)
                }
            },
            
            YAxes = new[]
            {
                new Axis
                {
                    Name = "Speed (m/s)",
                    TextSize = 12,
                    LabelsPaint = new SolidColorPaint(SKColors.LightGray),
                    MinLimit = 0,
                    MaxLimit = 350
                }
            },
            
            DrawMarginFrame = new DrawMarginFrame
            {
                Stroke = new SolidColorPaint(SKColors.Gray) { StrokeThickness = 1 }
            },
            
            // ATLAS-style cursor
            TooltipPosition = TooltipPosition.Hidden,
            PointerPressed += OnChartPointerPressed
        };
        
        Content = _chart;
    }
    
    private void OnChartPointerPressed(object sender, PointerEventArgs e)
    {
        // Update synchronized cursor position across all charts
        var xValue = _chart.ScaleUIToPixels(e.GetPosition(_chart)).X;
        UpdateCursorPosition(xValue);
        UpdateDataTable(xValue);
    }
}
